# yarn-inventory-app
An app to calculate the cost to sell and to keep check of inventory


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start
```
## Running the api

```bash
# development
$ cd laravel

# watch mode
$ php artisan serve
```
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test e2e

# test coverage
$ npm run test cov
```
